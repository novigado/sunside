# City Shadow Analyzer - UI Extension

This extension provides the user interface for the City Shadow Analyzer application.

## Features

- Location controls (latitude/longitude input)
- Time settings (current time or custom)
- Real-time sun position calculation and display
- Visual sun light positioning in 3D scene
- Test scene generation with simple buildings
- Shadow visualization using RTX rendering

## Usage

The UI window will automatically appear when the application launches. Use the controls to:

1. Set your location coordinates
2. Select the time (current or custom)
3. Click "Update Sun Position" to calculate and visualize
4. Use "Create Test Scene" to generate sample city buildings
5. Observe shadows in the viewport using RTX rendering
